Üdvözlünk a Next.js dokumentációjában!

Ez a **Kezdő lépések** rész segít létrehozni az első Next.js alkalmazásodat, és megismerkedhetsz az egyes projektekben használandó alapvető funkciókkal.

## Előzetes ismeretek

Dokumentációnk feltételez némi webfejlesztési jártasságot. Mielőtt belekezdenél, hasznos, ha magabiztosan ismered a következőket:

- HTML
- CSS
- JavaScript
- React

Ha még csak most ismerkedsz a Reacttel, vagy ha felfrissítenéd a tudásod, javasoljuk, hogy kezdd a [React Foundations kurzusunkkal](/learn/react-foundations), és a [Next.js Foundations kurzussal](/learn/dashboard-app), ahol a tanulás közben építhetsz alkalmazást.
